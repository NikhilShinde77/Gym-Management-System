
# Gym Management System

## Project Overview

The **Gym Management System** is a web-based application designed to manage gym operations digitally.
It helps manage **members, trainers, membership plans, attendance, and payments** efficiently.

This system provides three main panels:

* **Member Panel**
* **Trainer Panel**
* **Admin Panel**

The project aims to automate gym management and reduce manual work.

---

## Project Objectives

* Automate gym membership management.
* Track member attendance and subscription plans.
* Manage trainer assignments and workout schedules.
* Maintain payment records and generate reports.

---

## Features

### Member Panel

* Register and Login
* Choose Membership Plan
* View Workout Schedule
* Mark Attendance
* View Subscription Status
* View Payment History

### Trainer Panel

* Trainer Login
* View Assigned Members
* Create Workout Plans
* Update Member Progress
* Manage Training Schedule

### Admin Panel

* Manage Members and Trainers
* Create Membership Plans
* Monitor Attendance
* Generate Revenue Reports
* Maintain System Records

## Installation Steps

### 1. Clone the Repository

```
git clone https://github.com/NikhilShinde77/Gym-Management-System.git
```

### 2. Go to Project Folder

```
cd Gym-Management-System
```

### 3. Install Backend Dependencies

```
cd backend
npm install
```

### 4. Run Backend Server

```
node server.js
```

### 5. Install Frontend Dependencies

```
cd frontend
npm install
```

### 6. Run Frontend

```
npm start
```
